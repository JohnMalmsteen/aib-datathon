Datathon API

This page lists how to use our API in order to gain valuable insights into our customers data.

You may use this API to understand our customer demographics and how much of a "safe to spend balance" a customer has.

The data served on this API is an aggregation of five files given by AIB with over 5 million rows.
This project began at the AIB Datathon (hence the name) competition in Dublin on the 7th of Nov 2015.

Routes

HTTP GET, POST, PUT and DELETE operations are in use.



GET

/datathon/customer/:id

Example: /datathon/customer/1

Returns the customer object with given id

GET

/datathon/categories

Returns all categories and subcategories

DELETE

/datathon/customer/:id

Deactivates account



PUT

/datathon/customer/:id

Updates customer by id

Body: {payday: Number,county: String}

PUT

/datathon/customer/togglestatus/:id

Deactivate/Reactivate a customer account



POST

/datathon/customer

Adds a new customers

Body: {balance: Number, income: Number, payday: Number, age: Number, sex: String, county: String}

POST

/datathon/customer/add/transaction/:id

add a new transaction to the customers transaction list

Body: {category: String,subcategory: String,ammount: Number,type: String}

POST

/datathon/customer/add/rent/:id

add a new rent transaction to a customers

Body: {ammount: Number}
