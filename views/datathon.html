Customer Insights API

Welcome to the customer insights API.

This page lists how to use our API in order to gain valuable insights into customer demographics
and also how much of a "safe to spend balance" a customer has.


Routes

HTTP GET, POST, PUT and DELETE operations are in use.

They all return JSON objects.


GET

/datathon/customer/:id

Example: /datathon/customer/1

Returns the customer object with given id



PUT

/datathon/customer/:id

Updates customer information



POST

/datathon/customer

Adds a new customers



DELETE

/datathon/customer/:id

Deletes the customer object with given id
